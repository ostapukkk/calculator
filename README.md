Остапук Екатерина Никалаевна группа 20ИТ-2

Лабораторная работа №1

Окно приложения имеет фиксированный размер и не изменяемое.

Окно может быть закрыто с помощью кнопки Cloze(закрыть).

Окно может быть свернуто на панель задач с помошью кнопки  Свернуть.

Строка калткулятора содержит Иконку, Название и Кнопки.

Табло калькулятора 10-ти разрядное.

Ввод цифр производиттся нажатием клавиш на калькуляторе.

Клавиши арифметических действий нажимаются после первого числа.

Для выполнения ариф. действий необходимо ввести первый операнд, ввести знак арифметического действия, ввести второй операнд и 
нажать клавишу "=", чтобы получить результат или продолжить ввод, нажав знак ариф. операции для продолжения вычислений.
